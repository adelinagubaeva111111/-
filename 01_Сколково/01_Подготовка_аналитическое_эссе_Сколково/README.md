# Подготовка к аналитическому эссе (Сколково)

Здесь лежат пошаговые материалы на **5–7 дней**: шпаргалки, шаблоны для записей и упражнений по плану подготовки (риторика, тексты, банк аргументов, междисциплинарность, креатив, спринты под формат экзамена).
Раздел про эссе дополняет кейс-направление: аргументация и структурное мышление отсюда напрямую усиливают качество защиты на кейс-чемпионате.
