# 01_Сколково

Здесь собраны материалы по подготовке к задачам Сколково: аналитическое эссе, кейс-чемпионат и бизнес-тренировки.
Папка устроена так, чтобы быстро находить нужный режим подготовки и работать по понятной структуре.

## Подпапки

- `01_Подготовка_аналитическое_эссе_Сколково` — материалы по эссе и аргументации.
- `02_Кейс-Чемпионат` — конкурсные планы, правила поведения, рабочие чек-листы.
- `Бизнес` — система по решению кейсов, датасет, тренировки и промпты для презентаций.
