# Бизнес

Здесь рабочий контур для кейс-чемпионатов: от системы тренировки решений до промптов на структуру финальной презентации.
Раздел помогает быстро перейти от анализа кейса к защите перед жюри.

## Подпапки

- `01_Система_Кейсов` — рубрика, пайплайн, датасет и постмортемы тренировок.
- `02_Презентация_Промпт` — боевой промпт для сборки структуры презентации под 3-часовой формат.
