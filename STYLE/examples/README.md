# Примеры текстов

Здесь **образцы** удачных текстов в моем стиле (посты, сценарии, отчеты). Опирайся на них при генерации текстов «как я».

## Подпапки (как сейчас на диске)

- **`telegram/`** — посты личного телеграм-канала, примеры лежат в папке (`01_…`–`06_…`)  
- **`tiktok/`** — сценарии для стади-тиктока  
- **`повседневное/`** — внеконтентный стиль: рефлексии, ответы, запросы, отчёты. Сейчас лежат:
  - `01_ежедневная_рефлексия.md`
  - `02_еженедельный_отчет.md`
  - `03_ответ.md`
  - `04_запрос.md`
